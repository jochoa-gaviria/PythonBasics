# DataVisualizationWithPython

### First steps
###### Installing apps required.
- pip3 install justpy==0.1.5 
    library for building web apps and data visualization
- pip3 install pandas==1.2.2
    library for data analysis
- pip3 install pytz==2021.1
    library for datetime calculations between timezones
- pip3 install matplotlib==3.3.4
    library for quick data visualization
